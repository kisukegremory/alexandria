# Hardware Issues Guide

## Laptop Won't Turn On

1. Ensure the power cable is firmly connected at both ends
2. Try a different power outlet
3. If the battery is removable, remove it for 30 seconds then reinsert
4. Hold the power button for 10 seconds (hard reset)
5. If the power LED flashes an error pattern, note the number of flashes and contact IT

## Screen / Display Issues

### Black screen after login
- Press **Win + P** to cycle display output modes
- Connect an external monitor via HDMI/USB-C to determine if the issue is the panel or the GPU
- Update display drivers: Device Manager → Display Adapters → Update driver

### External monitor not detected
1. Check cable connections (try a different cable)
2. Right-click desktop → **Display Settings** → **Detect**
3. Try a different port on the laptop (HDMI vs USB-C)

### Screen flickering
- Disable hardware acceleration in your browser (Settings → Advanced)
- Update or rollback display drivers via Device Manager
- If flickering persists after driver changes, the panel or GPU may need replacement — open a ticket

## Keyboard / Mouse Issues

### Keys not responding
- Restart the laptop (rules out software hang)
- Check for debris under keys — use compressed air
- If using USB keyboard: try a different USB port; test on another computer
- Driver update: Device Manager → Keyboards → Update driver

### Mouse cursor frozen or jumping
- Unplug and replug the mouse
- Try a different surface (mouse pads work better than glass)
- For wireless mice: replace batteries or check the USB receiver dongle
- Laptop touchpad: Enable/disable with **Fn + F7** (or the touchpad icon key)

## Performance Issues (Slow Computer)

1. Restart the computer — clears memory leaks
2. Check Task Manager (**Ctrl + Shift + Esc**) for processes consuming high CPU or RAM
3. Run Windows Update — pending updates sometimes stall in the background
4. Check disk space: **This PC** → confirm C: drive has at least 10% free
5. Run **Disk Cleanup** (search in Start menu) to remove temp files

## USB / Peripheral Not Recognized

1. Try a different USB port
2. Unplug, wait 10 seconds, replug
3. Device Manager → **Action → Scan for hardware changes**
4. If a yellow exclamation mark appears, update the driver

## When to Open a Ticket

Open a support ticket if:
- Hardware physically damaged (cracked screen, broken hinge, liquid spill)
- Battery drains in under 2 hours under normal use
- Hard drive clicking noise (possible failure — back up data immediately)
- Issues persist after following the steps above

Include in your ticket: device model, serial number (on a sticker on the bottom of the device), and a description of the problem.
