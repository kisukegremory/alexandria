# Password Reset Guide

## Self-Service Reset (Recommended)

Most password issues can be resolved without contacting IT.

### Option 1 — IT Portal self-service
1. Go to `https://identity.techcorp.internal/reset`
2. Enter your employee email (e.g. `joao.silva@techcorp.com`)
3. Choose verification method: email code or authenticator app
4. Follow the prompts to set a new password
5. Password must be at least 12 characters with uppercase, lowercase, number, and symbol

### Option 2 — Windows login screen
On the Windows login screen, click **"Forgot my password"** below the password field. This works even if you're not connected to VPN.

### Option 3 — Manager reset
Your direct manager can trigger a temporary password reset via the HR portal at `https://hr.techcorp.internal/team`.

## Password Policy

| Requirement | Value |
|-------------|-------|
| Minimum length | 12 characters |
| Uppercase required | Yes |
| Number required | Yes |
| Special character required | Yes |
| Cannot reuse last | 10 passwords |
| Expires every | 90 days |

## Account Locked?

After 5 failed login attempts, your account locks for 30 minutes automatically. Wait 30 minutes and try again with the correct password.

If you need immediate access (urgent business reason), your manager can request an emergency unlock by emailing `it-urgent@techcorp.com` with your employee ID.

## MFA Issues

If your authenticator app is lost or showing wrong codes:
1. Contact IT Security at `security@techcorp.com` to reset your MFA enrollment
2. Provide government-issued ID for identity verification
3. A new QR code will be sent to your manager

## Still locked out?
Open a support ticket with your employee ID and a description of the issue.
