# Hello World!
Basicamente para iniciarmos um projeto em golang fazemos o uso do `go mod init 'nome/do/modulo'`, já com o prefixo com a ideia de onde ele será compartilhado, então costumeiramente veremos github.com/... com um grande link

Para baixar uma dependência que já está presente no código basta digita `go mod tidy`

Para rodar o programa basta estar na pasta em que os arquivos golang estão e rodar `go run .
